--- layout: post title: LiveValidator for ajaxified springframework categories: - Technology tags: [] status: publish type: post published: true meta: dsq_thread_id: '250173533' --- LiveValidator allows developers to turn their existing Validators into web2.0 ajax validators. Simply implement LiveValidator interface for any Validator implementation you want to ajaxify. That's it! Now your web-application is automatically validating user input using ajax and displaying any error message in a traditional spring:bind tags. From where can I download LiveValidator extension for SpringFramework? Nowhere.
